# Setup DC++ 

### MacOS

1. [Install Homebrew](https://brew.sh/) - skip if already present.
	- To check whether it is installed or not use the following command

		```sh
		command -v brew
		```

	- If the output is like follwoing then brew is installed!
		```sh
		~> command -v brew
		/opt/homebrew/bin/brew
		```
